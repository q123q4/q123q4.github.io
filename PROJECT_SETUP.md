### 🚀 启动项目

#### 方法1：使用脚本
- Windows: 双击 `start.bat`
- macOS/Linux: 运行 `./start.sh`

#### 方法2：手动启动
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

#### 访问地址
- 开发服务器：http://localhost:5173
- 预览生产版本：http://localhost:4173

### 🔧 自定义配置

#### 个性化内容
1. **个人信息**: 修改 `src/pages/Home.tsx` 和 `src/pages/About.tsx`
2. **作品项目**: 更新 `src/pages/Portfolio.tsx` 中的项目数据
3. **博客文章**: 编辑 `src/pages/Blog.tsx` 中的文章内容
4. **联系方式**: 修改 `src/pages/Contact.tsx` 中的联系信息
5. **服务内容**: 更新 `src/pages/Services.tsx` 中的服务描述

#### 样式定制
- **主色调**: 编辑 `tailwind.config.js` 中的颜色配置
- **字体**: 修改 `tailwind.config.js` 中的字体设置
- **动画**: 调整组件中的 Framer Motion 动画参数

#### 路由配置
- **添加页面**: 在 `src/App.tsx` 中添加新的路由
- **修改路径**: 更新路由配置中的路径设置

### 🌐 部署指南

#### Vercel（推荐）
1. 推送代码到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署完成

#### Netlify
1. 运行 `npm run build`
2. 上传 `dist` 文件夹
3. 配置重定向规则

#### 传统服务器
1. 运行 `npm run build`
2. 部署 `dist` 文件夹内容
3. 配置服务器支持 SPA 路由
