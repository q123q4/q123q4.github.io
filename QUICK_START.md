## 📲 移动设备访问

### 📱 在手机上访问
1. 确保手机和电脑在同一WiFi网络
2. 查看电脑IP地址：
   ```bash
   # Windows
   ipconfig
   
   # 找到 "IPv4 地址"，例如：192.168.1.100
   ```
3. 在手机浏览器访问：
   ```
   http://[您的IP地址]:8000
   ```
   例如：`http://192.168.1.100:8000`

## 🔧 启动问题解决

### 如果Python服务器无法启动
1. **检查Python安装**：
   ```bash
   python --version
   ```
   
2. **手动启动**：
   ```bash
   cd "c:/Users/王博/Desktop/个人主页"
   python start_server.py
   ```

3. **使用其他端口**：
   ```bash
   python -m http.server 3000
   ```
